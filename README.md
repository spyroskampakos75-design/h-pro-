# H-PRO - Hygiene Pro System

Progressive Web App για Σύστημα Ελέγχου Υγιεινής

## 🚀 Quick Start

1. Άνοιξε το link στο κινητό σου
2. Chrome → Μενού (⋮) → "Add to Home screen"
3. Done! Εικονίδιο στην αρχική οθόνη!

## ✨ Features

- 📱 Progressive Web App
- 🔒 Offline support
- 💾 localStorage data persistence
- 🎨 Professional UI
- ⚡ Fast & responsive

## 📦 One File Solution

Όλα embedded σε ένα αρχείο - No external dependencies!
